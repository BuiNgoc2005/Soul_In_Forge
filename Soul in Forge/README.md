# Soul In Forge – Unity Client
## 1. Mở project
- Mở Unity Hub
- Chọn "Open"
- Trỏ đến thư mục chứa project Unity (Soul In Forge)
---
## 2. Chạy game
- Mở scene **MainMenu**
  File: Assets/Scenes/MainMenu.unity
- Bấm nút **Play** trên thanh Unity
> ⚠ Không chạy trực tiếp ForgeScene vì đăng nhập sẽ không hoạt động.
---
## 3. Kiểm tra đăng nhập
Trong MainMenu:
- Chọn **Sign In** để tạo tài khoản mới  
- Chọn **Log In** để đăng nhập tài khoản đã có
Sau khi login thành công:
- Game tự chuyển sang **ForgeScene**
---
## 4. Dừng game
- Bấm **Play** lần nữa để dừng.
---
## 5. Lưu ý
- Luôn chạy từ **MainMenu** để đảm bảo hệ thống login hoạt động đúng.
- Không cần build game, chạy trực tiếp trong Unity Editor là được.
